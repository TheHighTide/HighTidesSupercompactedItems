# HighTidesSupercompactedItems
HighTide's Supercompacted Items is a mod for Subnautica that allows you to supercompress your items into supercompacted items making it more efficient to store large amounts of certain items.

### Requirements
For this mod to work in Subnautica, you need the following mods:
- [Nautilus](https://www.nexusmods.com/subnautica/mods/1262)
